<h3>Code</h3>

```

.requests {
  width: 350px;
  text-align: center;
  &:hover{
    box-shadow: 3px 3px 5px black;
    animation-duration: 200ms;
    animation-name: fadeIn;
  }
}

@keyframes fadeIn {
  0% {
    opacity: 0;
  }
  50% {
    opacity: 0.5;
  }
  100% {
    opacity: 1;
  }
}

```

This is a cool looking blink effect that occurs when user hover over a team.
